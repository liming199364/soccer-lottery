# Soccer Lottery (足彩分析助手)

⚽️ 足球分析与足彩预测 AI Skill（智能体技能） —— 基于历史对阵、伤停情报和热度降权，一句话给出方向与信心！
完美兼容 Trae Solo、Claude Code、OpenClaw 等主流 AI 平台。无需编写代码，使用自然语言即可轻松实现 AI 足彩分析专家。

## 它可以做什么
只需对 AI 说：「**今日红单**」或「**怎么买**」
  → 赛事全量抓取 → **多维情报采集** (H2H + 伤停核实) → **智能避热降权** (热门场次自动下调信心) → 输出方向与信心报告

赔率由你提供 —— 系统不主动检索赔率，你可以随时发截图或输入赔率数字，系统会据此做波动分析或串关计算。

其它常用说法（详见仓库根目录 `SKILL.md`）：
- **公众号 / 微信单篇**：提供 `mp.weixin.qq.com/s/...` 链接 → 可自动拉正文配图并读图，按场次还原 **胜平负 / 让球** 并标 **场次级逻辑矛盾**。
- **赔率截图模式**：上传多场 **胜平负 / 让球** 赔率与变盘截图，并说明 **以图为准** → 仅基于截图做 **波动分析**，**不会**再用 API / 联网赔率覆盖你的数字（除非你明确要求对照）。

## 核心分析模型

### 1. 智能避热 (Hot Match De-weighting)
系统会自动识别「顶级热门」场次（皇马、巴萨、曼城、PSG、拜仁、利物浦、阿森纳等）。对于高热比赛，系统会**自动下调 10-15% 的信心值**，信心上限 75%，并强制输出风险提示。

### 2. 信心加权模型
综合三大维度：
- **伤停情报 (40%)**：核心球员缺阵对战术体系的打击程度。
- **H2H 历史 + 近况 (40%)**：近期交锋胜负比、联赛状态（连胜/连败/主客场差异）。
- **热度降权 (20%)**：热门场次自动降权，冷门联赛维持正常信心上限。

### 3. 赔率由用户提供
系统不主动检索赔率。当用户提供赔率（截图或文字）时，系统会结合赔率走势进一步修正信心和方向，并可给出串关组合。

## 分析流程

### 🔄 数据获取
```
1. Football-Data.org API (赛事数据)
   └─ 获取今日对阵列表、H2H 历史交锋

2. WebSearch 联网补全
   └─ 核实伤停名单、近期状态
   └─ 覆盖 API 未支持的联赛（日职、韩职、中超等）
```

### 🧠 分析流程
```
1. H2H 历史分析
   └─ 近N场交锋胜负比
   └─ 主客场差异

2. 伤停情报核实
   └─ 核心球员缺阵评估
   └─ 对战术体系影响程度

3. 热度评估与降权
   └─ 识别高热场次（豪门对决）
   └─ 自动下调信心值 10-15%

4. 信心指数输出
   └─ 综合三维度加权，输出方向与信心
```

### 📊 支持的联赛

1. **欧洲顶级联赛 (API 驱动)**：
   - 英超、西甲、德甲、意甲、法甲、欧冠、欧联、葡超等。
   - 数据源：Football-Data.org。

2. **亚洲及其他联赛 (WebSearch 补全)**：
   - 日职联 (J-League)、韩职联 (K-League)、中超、亚冠等。
   - 荷甲、比甲、瑞典超、挪超、美职联等。
   - 通过 WebSearch 实时校验对阵与伤停信息。

## 配置
在 `config.yaml` 中配置你的数据源 API Key：
```yaml
api:
  football_data:
    key: "YOUR_FOOTBALL_DATA_API_KEY"  # Get from https://www.football-data.org/
```

### 如何获取 API Key

**Football-Data.org (赛事数据)**
- 访问 [football-data.org 官网](https://www.football-data.org/)
- 选择免费的 **"Free Tier"**（支持五大联赛和欧冠等顶级赛事）
- 注册后 API Token 会发送到你的邮箱

## 功能特性

### 🏆 输出类型
- **方向判断**：胜（主胜）、平（平局）、负（客胜）
- **信心指数**：基于 H2H + 伤停 + 热度降权的综合评分

### 📊 输出内容
- 表格一览式分析（方向 + 信心 + 热度 + 关键因素 + 伤停 + 风险）
- 按信心从高到低排列
- 提示用户提供赔率以进一步做串关推荐

### 🌍 球队名称翻译
自动翻译球队名称为中文，支持常用球队特殊名称映射（如 Arsenal → 阿森纳）

## 辅助功能
除了全流程分析，你还可以直接进行单项查询：
- **单场分析**：`python3 scripts/analyzer.py --match <id>`
- **微信文章 HTML**：`python3 scripts/parse_wechat_article_html.py --url '<mp.weixin.qq.com 链接>'`
- **微信正文配图**：`python3 scripts/fetch_wechat_article_images.py --url '<同一链接>'`
